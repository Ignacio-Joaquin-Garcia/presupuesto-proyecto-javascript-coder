# Friends Budget

> *Creado por Ignacio Garcia*

Aplicacion Web dinamica No Responsive que permite contabilizar y hacer estadisticas, generando graficos en tiempo real acerca de datos ingresados. Esta especificamente ambientada para situaciones como: juntadas entre amigos donde hace falta contabilizar lo que gasta cada uno para una correcta divisiÃ³n del gasto.
